# USB-TYPE-C
USB Type-C Device board

This project is made with [KiCAD](https://kicad.org).

# Features

* 350mA Resettable Fuse
* Configuration Channel(CC) -> Pull downed with 5.1kΩ Resistor
* 10μF Capacitor
